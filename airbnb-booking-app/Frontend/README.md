# Airbnb Booking App — Frontend

React frontend for the Airbnb Booking App, built with **Vite**, **React 18**, **React Router v7**, and **Tailwind CSS**.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

> **Note:** The backend server must be running on `http://localhost:3000` for API calls to work.

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start Vite dev server with HMR |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build locally |
| `npm run lint` | Run ESLint checks |

See the [root README](../README.md) for full project documentation.
